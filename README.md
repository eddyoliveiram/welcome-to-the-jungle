# WelcomeToTheJungle

> Projeto Django de exemplo: blog com usuários, posts e comentários.

Instruções rápidas:

1. Crie um ambiente virtual e instale dependências:

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

2. Rode migrações e o servidor:

```bash
python manage.py migrate
python manage.py runserver
```

Usuários iniciais e dados são criados por migração de dados.
